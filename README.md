# IoT-Based Smart Home Network using Cisco Packet Tracer

## Project Overview
This project demonstrates an Internet of Things (IoT) based Smart Home simulation using Cisco Packet Tracer. The network connects multiple smart devices through a Home Gateway, allowing users to monitor and control appliances remotely.

## Objectives
- Simulate a smart home environment.
- Connect IoT devices to a wireless network.
- Demonstrate remote monitoring and control.
- Understand IoT communication within a local network.

## Components Used
- Cisco Home Gateway
- Wireless Router
- PC/Laptop
- Smartphone
- Smart Light
- Smart Fan
- Smart Door
- Temperature Sensor
- Smoke Detector
- IoT Server (if applicable)

## Working
1. The Home Gateway provides wireless connectivity to all IoT devices.
2. Each smart device registers with the Home Gateway.
3. Devices receive IP addresses automatically using DHCP.
4. Users log in through a PC or smartphone.
5. Sensor data is continuously transmitted to the gateway.
6. Users can remotely monitor and control connected devices.
7. Automation rules can trigger actions based on sensor readings.

## Features
- Wireless IoT communication
- Automatic IP addressing
- Device monitoring
- Remote appliance control
- Smart automation

## Applications
- Smart Homes
- Smart Offices
- Industrial Automation
- Healthcare Monitoring

## Software Used
- Cisco Packet Tracer

## Conclusion
This project demonstrates the basic implementation of IoT networking and illustrates how smart devices communicate through a centralized gateway for automation and remote control.
